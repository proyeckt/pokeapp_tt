## ğŸ—ï¸ Architecture

This project follows **Clean Architecture** principles for maintainability and scalability:

- **Presentation Layer**:  
- Widgets and feature-based UI modules
- Material Design conformity
- Navigation with GoRouter

- **State Management**:  
- **Riverpod + Annotations** for reactive, typed state
- Clean separation of logic from UI

- **Domain & Data Layers**:  
- **Freezed** for immutable models, simple yet powerful
- **Dio** for all network operations
